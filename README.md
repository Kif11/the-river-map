## THE RIVER MAP

Map for volunteering community to discover places to stay.

![The River Demo](https://user-images.githubusercontent.com/8003487/38347689-86403714-3862-11e8-992f-254fd50b22f9.gif)

To clone, build and run:
```
git clone https://github.com/Kif11/the-river-map
cd the-river-map
yarn install
yarn start
```
